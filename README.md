"# AI-Powered Web Application This is a collaborative Spring Boot web app with AI features, using JTE template engine." 

## How to Use

1. Run the Spring Boot application locally.
2. Open the web app in your browser.
3. Enter text into the input form and submit to get an AI-generated response.
4. **Input Validation:** Make sure your input is not empty and less than 500 characters. If your input is invalid, an error message will be displayed.
